THESIS PROJECT · 2025
PhiNex
Platform PYNQ-Z2 ARM-FPGA Board
Category Cybersecurity Gateway
Status Completed ✓
Role Lead Developer
PhiNex is a phishing detection security gateway built on the PYNQ-Z2 ARM-FPGA development board. It leverages edge inference with a scikit-learn MLPClassifier converted to TensorFlow Lite to detect phishing URLs in real time. The system features multi-modal alerts including WebSocket push notifications, Telegram integration, and audio/LED indicators — all managed through a responsive web dashboard.

INTERNSHIP PROJECT · 2026
MRF Digitalization
Client Manufacturing Company QA Dept.
Category Web Dashboard · QA System
Status In Progress ⚡
Role Full Stack Developer
A web-based dashboard designed to digitalize the Modification Request Form (MRF) process for a manufacturing company's QA department. The system eliminates paper-based workflows that were creating mountains of physical paperwork. Features role-based access control, a two-part digital MRF form, defect tracking, and a real-time QA dashboard for management visibility.
